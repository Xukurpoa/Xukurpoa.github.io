Chain Reaction

A Java based implementation of the multiplayer game Chain Reaction. Two versions were made; One of which implemented local multiplayer on the same device, which the latter utilized a networking API in order to offer asynchronous multiplayer. The networked model was based on a hub wheel paradigm, where one player would act as the central controller for the game, exchanging messages with all the players

Summer 2019

Languages/Technologies used: Java, NetGame API
